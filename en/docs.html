Documentation

PhraseLauncher is a simple launcher that allows you to paste fixed phrases quickly using only your keyboard.

4. Register Phrases

You can display the launcher by pressing Ctrl twice in a row (the key type and number of presses can be changed in the settings) or by using Ctrl + Shift + O.
